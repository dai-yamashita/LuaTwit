Class future

Future object. It contains the result of an asynchronous API call.
future:peek ()
Checks (non-blocking) and returns the value of the future if it's ready.

Returns:

  1. true if the value is ready, otherwise false.
  2. List of return values from the API call.
future:wait ()
Waits (blocks) until the value of the future is ready.

Returns:

    List of return values from the API call.

Class service

HTTP service object. Executes HTTP requests on background (with curl.multi).
service.new ()
Creates a new async http client.

Returns:

    New instance of the async client.
service:set_conn_limits (total_conn, host_conn)
Sets the connection limits.

Parameters:

  • total_conn Maximum number of total connections (default unlimited).
  • host_conn Maximum number of connections per host (default unlimited).
service:data_available ()
Checks if there is data pending to be received.

Returns:

    If there is data available, the current number of unfinished requests. Otherwise nil.
service:http_request (request, filter)
Performs an asynchronous HTTP request.

Parameters:

  • request Request objected created by curl.easy.
  • filter Function to be called on the result data.

Returns:

    future object with the result.
